Qu es una base de datos?

Es la coleccin de informacin estructurada que puede ser facilmente consultada, manipulada, y actualizada

Las bases de datos estan organizadas en filas, columnas, y tablas y suele utilizarse la indexacin para facilitar el acceso a la informacin relevante.

Flat database

Base sencilla que guarda toda la informacin en una sola tabla o lista.

  • Puede contener varios campos.
  • Proclive a tener datos duplicados.
  • Para juntar dos flat files necesitas copiar info de A a B.
  • No hay automatizacin entre flat files. Un cambio en A no afecta a B.
  • Ofrecen funcionalidad en el almacenaje, manipulacin, y acceso de informacin.

Relational database

Incorpora multiples tablas con mtodos informticos.

  • Puede contener varios campos como las flat database.
  • Resuelve el problema de tener datos duplicados por observacin.
  • La base A y B interactuan por lo que no necesitas copiar A en B.
  • Puede haber automatizacin entre databases.
  • Almacena, manipula, y accede a informacin MUCHO mas comleja y de manera eficiente.
  • Al ingresar nueva informacin o modificar una base de datos se debe considerar la compatibilidad con las otras bases.

  • No hagas clculos o conversiones antes de meter los datos.
